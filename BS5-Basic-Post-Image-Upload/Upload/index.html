Setup
For examples to work: Set up the demo database table by running the following sql query on your test db in PHPMyAdmin.
Code:

CREATE TABLE IF NOT EXISTS `posts` (
`post_id` INT(11) PRIMARY KEY AUTO_INCREMENT,
`post_image` TEXT NOT NULL
);

INSERT INTO `posts` VALUES('1', '1666061875.jpg');
In example connection file: db.php find and edit the db connections.
Code:

// Database connection details
$mysqli = mysqli_connect('localhost', 'db_user', 'db_password', 'db_name');
Final notes:

You then would visit example pages links above or below and begin future development edits and improvements from there. This simple PHP / MySQli snippet is simply shared "AS IS" in the hopes that it can be then further used as a starting base point for development.
